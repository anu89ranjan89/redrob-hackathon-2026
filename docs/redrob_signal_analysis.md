Divide signals into 3 categories
1. Recruitability Signals

These tell whether candidate is likely to respond and join.

Examples:

open_to_work_flag
recruiter_response_rate
notice_period_days
offer_acceptance_rate

Meaning:

Open to work = easier to hire
High response rate = replies to recruiters
Low notice period = can join quickly
High offer acceptance = likely to accept offer


2. Activity Signals

These tell whether profile is active.

Examples:

last_active_date
applications_submitted_30d
github_activity_score
search_appearance_30d
saved_by_recruiters_30d

Meaning:

Recently active candidate
Applying to jobs
Active on GitHub
Appearing in recruiter searches


3. Trust Signals

These tell whether profile is trustworthy.

Examples:

verified_email
verified_phone
linkedin_connected
profile_completeness_score

Meaning:

Verified profile
Complete profile
Less likely to be fake
