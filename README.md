# 🚀 Redrob Hackathon 2026

## AI-Powered Candidate Ranking System

An intelligent hiring assistant that ranks candidates based on:

- Technical Fit
- Recruitability
- Activity
- Trust

while detecting:

- Honeypot Profiles
- Skill Inflation
- Timeline Anomalies
- Profile Contradictions

---

## 🎯 Problem Statement

Given a Job Description (JD) and candidate profiles, identify and rank the most relevant candidates while providing explainable reasoning for every recommendation.

---

## 💡 Our Solution

We use a Hybrid Ranking Framework that combines:

### 1. Technical Fit (50%)
- Skill Matching
- Retrieval Systems Experience
- Ranking Systems Experience
- Recommendation Systems Experience
- Production ML Experience

### 2. Recruitability (25%)
- Recruiter Response Rate
- Notice Period
- Open To Work Status
- Offer Acceptance Rate

### 3. Activity (15%)
- Last Active Date
- Applications Submitted
- Recruiter Search Appearances
- Recruiter Interest

### 4. Trust (10%)
- Verified Email
- Verified Phone
- LinkedIn Connected
- Profile Completeness

---

## ⭐ Key Features

### Explainable AI Ranking

Instead of only showing scores, we explain:

- Why a candidate was selected
- Supporting evidence
- Recruitability indicators
- Potential concerns

Example:

Rank #4

Technical Fit: 92

Recruitability: 88

Activity: 91

Trust: 95

Risk Level: LOW

Reasons:
- Built recommendation systems
- Product company experience
- Active profile
- High recruiter response rate

Concern:
- Notice period 60 days

---

### Candidate Credibility Score

Measures profile trustworthiness using:

- Verification signals
- Timeline consistency
- Skill evidence
- Profile completeness

---

### Honeypot Detection Framework

Detects:

- Skill Inflation
- Timeline Anomalies
- Profile Contradictions
- Activity Mismatches
- Duplicate Profiles
